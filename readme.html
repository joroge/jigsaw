üß© Overview

The Sliding Puzzle Game is an interactive web-based puzzle game where players can upload their own images and solve them as sliding puzzles. This project combines classic puzzle mechanics with modern web technologies to create an engaging user experience.

Responsive Design

Game Controls

Easy-to-use shuffle, reset, and difficulty selection controls.

üöÄ How to Use

Upload an Image

Click the upload area or drag and drop an image file to create your custom puzzle.

Select Difficulty

Choose from 3√ó3 (easy) to 6√ó6 (challenging) grid sizes based on your preference.

Start Playing

Click the "Shuffle Puzzle" button to randomize the tiles and begin the game.

Solve the Puzzle

Click tiles adjacent to the empty space to move them and reconstruct the original image.

Track Progress

Monitor your moves and time in the statistics panel as you solve the puzzle.

Celebrate

Enjoy the confetti animation when you successfully complete the puzzle!

  • Timer and move counter functionality
  • Drag and drop file upload support

Game Mechanics

  • The puzzle consists of a grid with one empty space
  • Players move adjacent tiles into the empty space
  • The goal is to reconstruct the original image
  • The game tracks the number of moves and completion time
  • Puzzle is automatically checked for completion after each move

Customization

The game can be easily customized by modifying the CSS variables in the :root selector:

:root {
    --primary-color: #4a6fa5;
    --secondary-color: #6b8cbc;
    --accent-color: #ff6b6b;
    /* Add your own color scheme */
}
